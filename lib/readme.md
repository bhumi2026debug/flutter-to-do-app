# Flutter To-Do App

A simple To-Do List application built using Flutter.

## Features

- Add Tasks
- Delete Tasks
- Mark Tasks Complete
- Yellow Theme UI

## Concepts Used

- StatefulWidget
- setState()
- ListView.builder
- TextEditingController
- Custom Widgets
- Constructor Arguments
- Material Design
